## Spesifikasi
- Server OS: Windows 11
- Web Server: Apache
- PHP: 8.2.12
- Database: MySQL 8.0
- PHP Extensions: BCMath, Ctype, Fileinfo, JSON, Mbstring, OpenSSL, PDO, Tokenizer, XML

## How To Run
- Clone repository ke folder lokal
- Buat database **db_mahasiswabaru** di server MySQL localhost
- Jalankan command:
```
composer install
php artisan migrate
php artisan db:seed
php artisan serve

npm install
npm run dev
```
